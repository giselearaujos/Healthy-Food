## Instalação e uso

```bash
# Abra um terminal e copie este repositório com o comando
git clone https://github.com/3ddybr/healthy_food.git
# ou use a opção de download.

# Entre na pasta web com
cd healthy_food

# Instale as dependências
npm install

# Rode a aplicação
npm rum dev
```
